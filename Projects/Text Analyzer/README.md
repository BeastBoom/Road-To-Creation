# Text Analyzer CLI

Reads a `.txt` file, removes common stopwords, and outputs the top 10 most frequent words to `word_counts.json`.

## How to run locally
```bash
cd "Projects/Text Analyzer"
python text_analyzer.py
```
Expects a file named `file.txt` in the same folder. Writes `word_counts.json` in the same folder.
